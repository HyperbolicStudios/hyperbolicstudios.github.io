Bikewheel

No, these aren't pictures.

A true test of Vectorworks rendering potential. An exhausting 7-hour render brings this spectacular model of a bike wheel. The challenge of this project was to create a bike wheel using "sweep" and "array" techniques; instead of modelling each spoke individually, one spoke is modelled, and then duplicated in a cicular pattern.
